jQuery Scroll Table
====================
By: Spencer Neese [https://github.com/th3uiguy/](https://github.com/th3uiguy/)   
Version: 0.5.2   
Requires: jQuery UI 1.7+ and jQuery 1.3.2+   
Demo: [http://jsfiddle.net/th3uiguy/Yahvv/](http://jsfiddle.net/th3uiguy/Yahvv/)   
Git: [https://github.com/th3uiguy/jquery-scrolltable.git](https://github.com/th3uiguy/jquery-scrolltable.git)   


Description
---------------------
Simple jQuery compatible scroll table plugin.




Example
---------------------
```js
$('table.scrollTable').scrolltable({
	stripe: true,
	oddClass: 'odd'
});
```



Options
---------------------
#### maxHeight ####
*type: Integer*   
*default: 300*


#### setWidths ####
*type: Boolean*   
*default: true*


#### stripe ####
*type: Boolean*   
*default: false*


#### oddClass ####
*type: String*   
*default: "st-tr-odd"*


#### evenClass ####
*type: String*   
*default: "st-tr-even"*


#### firstClass ####
*type: String*   
*default: "st-tr-first"*


#### lastClass ####
*type: String*   
*default: "st-tr-last"*
